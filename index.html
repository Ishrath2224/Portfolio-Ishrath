Projects

Student Portal System

A complete student portal built using PHP and MySQL for the backend, and HTML, CSS, Bootstrap for the frontend. It allows students to register with full details, log in, view and edit their profile, and receive admin notices.  On the homepage, users can register with personal, academic, family, and payment details. Login is secured using first name and DOB. Upon logging in, students access a personalized dashboard with profile controls and a real-time notice board. Admin functionality is protected: only the portal owner can register admins using a secret key. Admins log in using email and password to access their dashboard where they can view student and admin records, and manage notices with status controls (activate/deactivate). The system also includes dynamic navigation bars showing user/admin names and profile images, enhancing personalization and user experience.
